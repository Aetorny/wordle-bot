# Wordle Bot — Веб-интерфейс

![Web UI screenshot](readme/web-ui.png)

Это локальный веб-интерфейс для алгоритма по угадыванию слов, который помогает выбирать наиболее вероятные слова в игре Wordle по ходу игры.

## Что делает проект
 - Предлагает список вероятных слов и вероятность для каждого варианта.
 - Позволяет вводить слова и тут же окрашивать их.
 - Поддерживает разную длину слова (4–11 букв) для русского языка.
 - Поддерживает класичесскую игру в wordle на английском с длиной 5.
 - Поддерживает быструю окраску букв с помощью клавиш (по умолчанию 1/2/3 → серый/жёлтый/зелёный).
 - Можно исключать слово из подсказок временно или навсегда (удаление из банка слов).

## Основные файлы
 - `web_ui.py` — standalone Flask-сервер, реализующий web ui.
 - `main.py` - запуск консоль версии программы, не требует flask.
 - `ui.py` — главный движок анализа (алгоритм/логика бота).
 - `analyze.py` — функция(и) для анализа и подсчёта вероятностей.
 - `words_uploader.py` — загрузка/подготовка банок слов (англ/рус).
 - `word banks/` — текстовые файлы с базами слов (например `english.txt`, `russian5.txt` и т.д.).

## Требования
 - Тестировалось на Python 3.14
 - Зависимости перечислены в `requirements.txt`
   - `Flask` необходим только для web версии программы
   - `pymorphy3` необходим только для повторного разделения русских слов в standalone файле `words_splitter.py`

## Установка и запуск (Windows)
1. Установка репозитория

```powershell
git clone https://github.com/Aetorny/wordle-bot
cd .\wordle-bot\
```

2. Опциональная установка виртуального окружения

```powershell
python -m venv .venv
.\.venv\Scripts\Activate
```

3. Установите зависимости:

```powershell
pip install flask
```

4. Запустите веб-сервер:

```powershell
python web_ui.py
```

5. Откройте в браузере: `http://127.0.0.1:5000/`.
